# سَعْي — منصة العمل الحر

منصة بسيطة وأنيقة لربط المستقلين بالعملاء، مستوحاة من تصميم خمسات ومواقع الفريلانس الشهيرة.

## الميزات
- تصميم متجاوب وأنيق مع ألوان محايدة.
- شرائح عرض (Slider) للخدمات.
- نماذج تسجيل آمنة عبر Formspree.
- قسم إحصائيات وشهادات عملاء.
- دعم كامل للعربية (RTL).

## كيفية التشغيل
1. قم بتنزيل الملفات (index.html, styles.css, script.js, images/).
2. افتح `index.html` في أي متصفح ويب (مثل Chrome أو Firefox).
3. للنشر: ارفع المجلد إلى GitHub Pages أو Netlify.

## المتطلبات
- متصفح حديث يدعم HTML5 وCSS3.
- اتصال إنترنت لتحميل الخطوط والأيقونات.

## التخصيص
- استبدل "your-form-id" في النماذج برابط Formspree الحقيقي (من formspree.io).
- أضف صور حقيقية في مجلد "images" للسليدير (مثل slide1.jpg).
